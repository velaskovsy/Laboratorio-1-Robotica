## Descripción del Laboratorio
El laboratorio tiene como objetivo el comprender y simular el comportamiento cinematico de un robot móvil diferencial usando el simulador Webots. La meta es entender como las velocidades de las ruedas izquierda y derecha determinan la trayectoria y movimiento del robot.

Se implementará un controlador en python para manipular los actuadores y observar diferentes tipos de desplazamiento: movimiento rectilíneo, trayectorias curvas y rotación sobre su propio eje.
